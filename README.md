# AI Investment Masters Library: O'Neil's Style

## 프로젝트 개요
이 프로젝트는 전 세계 유명 투자 대가들의 투자 철학을 학습한 AI 어시스턴트를 웹페이지 형태로 제공하는 것을 목표로 합니다. 현재 'O'Neil's Style' 어시스턴트 웹페이지의 초기 UI/UX 및 핵심 프론트엔드 구현이 완료되었습니다.

## 현재 진행 상황

### 1. 기술 스택
*   **프레임워크:** React.js
*   **스타일링:** Tailwind CSS
*   **빌드 도구:** Vite
*   **아이콘:** React Icons (`react-icons`)
*   **배포:** GitHub Pages (`gh-pages`)

### 2. UI/UX 디자인 컨셉: "The Professional's Desk"
*   **전체적인 분위기:** 깔끔하고 모던하며, 정보를 직관적으로 전달하는 최신 웹페이지 디자인. 다크 모드 기본 테마.
*   **레이아웃:**
    *   **헤더:** 프로젝트 로고("AI Investment Masters Library")와 간단한 네비게이션 ("About", "Contact", "Settings").
    *   **좌측 사이드바 (책장 역할):** 전체 페이지 너비의 약 20%를 차지하며, "My Masters" 아래에 가상 '책' 아이콘들이 세로로 나열됩니다. 현재 "O'Neil's Style" 책이 클릭 가능하며, 마우스 오버 시 확대 효과가 적용됩니다. (다른 대가 책들은 현재 비활성화 상태입니다.)
    *   **메인 컨텐츠 영역:** 좌측 사이드바를 제외한 80% 공간을 차지하며, 선택된 '책'에 따라 내용이 동적으로 변경됩니다.

### 3. 'O'Neil's Style' 챗봇 인터페이스 구현
"O'Neil's Style" 책 클릭 시 메인 컨텐츠 영역은 다음과 같이 구성됩니다:

*   **상단:** "O'Neil's Style Assistant" 제목과 "윌리엄 오닐의 CAN SLIM 원칙에 기반한 투자 분석" 설명.
*   **중앙 메인 영역 (분할):**
    *   **왼쪽 절반 (약 60%): '차트 분석 창'**
        *   주식 차트가 표시될 공간 (모의 UI).
        *   종목 검색창과 "일봉", "주봉" 등의 시간대 선택 버튼 포함.
    *   **오른쪽 절반 (약 40%): '기술적 지표 분석 사이드바'**
        *   차트 관련 기술적 지표(MACD, RSI 등) 분석 내용이 텍스트로 표시될 공간 (모의 UI).
*   **하단 영역 (탭 전환): '정보/채팅 통합 창'**
    *   세 가지 탭으로 구성: `[ 채팅 ]` `[ 뉴스 ]` `[ 재무 ]`
    *   **`[ 채팅 ]` 탭:** 사용자-AI 대화 채팅창 (버블 형태, 메시지 입력 및 전송).
    *   **`[ 뉴스 ]` 탭:** 종목/시장 관련 최신 뉴스 기사 목록 표시 공간 (모의 UI).
    *   **`[ 재무 ]` 탭:** 종목의 주요 재무제표 요약 정보 표시 공간 (모의 UI).

### 4. 코드 개선 및 배포
*   **파일 확장자 수정:** JSX 문법을 사용하는 컴포넌트 파일들의 확장자를 `.js`에서 `.jsx`로 변경하고, 관련 import 경로를 모두 업데이트하여 빌드 오류 및 렌더링 문제를 해결했습니다.
*   **GitHub Pages 배포 설정:**
    *   `gh-pages` 라이브러리를 설치했습니다.
    *   `package.json`에 GitHub Pages 호스팅을 위한 `homepage`, `predeploy`, `deploy` 스크립트를 추가했습니다.
    *   `vite.config.js`에 GitHub Pages 서빙을 위한 `base` 경로를 설정했습니다.
*   **GitHub Pages 배포 완료:** `npm run deploy` 명령을 통해 빌드된 프로젝트가 GitHub Pages에 성공적으로 배포되었습니다.

## 애플리케이션 실행 방법

1.  프로젝트 루트 디렉토리에서 터미널을 엽니다.
2.  다음 명령어를 실행하여 개발 서버를 시작합니다:
    ```bash
    npm run dev
    ```
3.  터미널에 표시되는 URL (일반적으로 `http://localhost:5173/`)을 웹 브라우저에서 엽니다.
